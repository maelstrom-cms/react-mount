## Install

[npm][]:

```sh
npm install react-mount
```

[yarn][]:

```sh
yarn add react-mount
```


## Usage

```js
const ReactMount = require('react-mount');

const reactMount = new ReactMount();

console.log(reactMount.renderName());
// script
```
